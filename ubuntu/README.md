# Ubuntu Setup Script
The script ubuntu\_18.04+\_setup.sh does the following:
- Updates Ubuntu to the latest packages
- installs git
- installs vim
- installs curl
- installs Google Chrome
- installs the JDK (JDK 11 by default)
- installs maven
- installs mariadb
- installs IntelliJ Idea
- installs VS Code
- installs Postman
- installs Slack
